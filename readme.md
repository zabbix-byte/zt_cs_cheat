## Usage
#### **WALL options**
- radar: True/False
- color: (r,g,a)

#### **AIM Options**
- conf_random: *5 to 25*
- conf_spotted: *True/False*
- conf_baim: *True/False*
- conf_aim_fov: *True/False*
- conf_smooth: *True/False*
- conf_aimtrcs: *True/False*
- conf_sens: *0.5 to 1.5*

#### **KEYS**
- **WALL**
  - 7 > START WALL
  - 7 > STOP WALL
- **BHOP**
  - 8 > ENTER TO BHOP
  - 8 > EXIT BHOP

- **AIM**
  - 9 > ENTER TO AIM
  - 9 > EXIT AIM

- **EXIT**
  - 0 > EXIT
